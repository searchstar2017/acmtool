java函数是值传递。当参数是引用时传递的是引用的值。
String name = "qjs"
name是一个引用。
static void func(String name) {
	name = "lalala"
}
这个函数返回后name的内容不变，因为时拷贝了引用的值。而name = "lalala"使得形参name指向了另一个String。

public static final int MAXN = 5010;

System.out.println(String.format("%.5f", ans));
